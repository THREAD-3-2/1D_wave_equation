1D wave equation Documentation¶

1D_wave_equation is a MATLAB code for the dynamics of a 1-dimensional wave equation. It consists in a second-order PDE for the description of waves, in this particular case transverse waves on a massive string. The trapezoidal quadrature rule is used to approximate the integral of the Lagrangian function in both time and space.
